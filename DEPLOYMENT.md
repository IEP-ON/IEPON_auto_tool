# 나이스 자동입력 확장 프로그램 배포 가이드

## 1. 다른 브라우저에서 사용하기 (개발자 모드)
이 확장 프로그램은 Chromium 기반 브라우저(Chrome, Edge, Brave, Whale, Opera)에서 모두 작동합니다. 배포 전에도 아래 방법으로 바로 사용할 수 있습니다.

### Microsoft Edge (엣지)
1. 주소창에 `edge://extensions` 입력
2. 좌측 하단 **'개발자 모드'** 켜기
3. **'압축 풀린 확장 로드'** 클릭
4. 프로젝트 폴더(`IEPON_auto_tool-main`) 선택

### Brave (브레이브)
1. 주소창에 `brave://extensions` 입력
2. 우측 상단 **'개발자 모드'** 켜기
3. **'압축해제된 확장 프로그램을 로드합니다'** 클릭
4. 프로젝트 폴더 선택

### Naver Whale (웨일)
1. 주소창에 `whale://extensions` 입력
2. 우측 상단 **'개발자 모드'** 켜기
3. **'압축해제된 확장앱 설치'** 클릭
4. 프로젝트 폴더 선택

---

## 2. 크롬 웹 스토어 배포하기 (정식 출시)
다른 사용자들이 쉽게 설치하게 하려면 스토어에 올려야 합니다.

### 1단계: 준비물
*   **Google 계정**
*   **등록비 $5** (해외 결제 가능 카드, 평생 1회)
*   **ZIP 파일**: 프로젝트 폴더 안의 파일들을 선택해서 `archive.zip`으로 압축
    *   *주의: 폴더 자체를 압축하지 말고, 폴더 안에 들어가서 `manifest.json`, `popup`, `content` 등을 드래그해서 압축하세요.*
*   **스크린샷**: 1280x800 픽셀 크기의 스크린샷 1장 이상
*   **아이콘**: 128x128 픽셀 아이콘 파일 (이미 `icons/icon128.png`가 있음)

### 2단계: 개발자 등록
1. [Chrome 웹 스토어 개발자 대시보드](https://chrome.google.com/webstore/dev/dashboard) 접속
2. 약관 동의 및 $5 결제

### 3단계: 업로드 및 정보 입력
1. 대시보드 우측 상단 **[+ 새 항목]** 클릭
2. 준비한 **ZIP 파일** 업로드
3. **스토어 등록정보** 작성:
    *   **설명**: 프로그램 기능 상세 설명
    *   **카테고리**: 생산성 또는 도구
    *   **스크린샷**: 준비한 이미지 업로드
4. **개인정보처리방침**:
    *   이 프로그램은 개인정보를 수집/전송하지 않고 로컬에서만 작동하므로, 설명란에 "이 확장 프로그램은 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다."라고 명시하면 됩니다. (구글 폼 등을 이용해 간단한 방침 페이지를 만들어 링크를 걸 수도 있습니다)

### 4단계: 게시
1. 모든 정보 입력 후 우측 상단 **[검토를 위해 제출]** 클릭
2. 심사는 보통 **24시간 ~ 48시간** 소요됩니다.
3. 심사가 통과되면 스토어 링크가 생성되고 누구나 다운로드할 수 있습니다.

## 3. 팁
*   **버전 관리**: 기능을 수정하고 다시 올릴 때는 `manifest.json` 파일의 `"version": "1.0.0"` 숫자를 올려야 합니다 (예: `1.0.1`).
*   **엣지 스토어**: [Microsoft Partner Center](https://partner.microsoft.com/dashboard/microsoftedge)에도 개발자 등록을 하면 엣지 전용 스토어에도 올릴 수 있습니다. (무료)
