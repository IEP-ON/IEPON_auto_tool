## 📌 소개

**특수교사 선생님들의 업무를 지원합니다**

개별화교육계획(IEP) 작성, 매번 반복되는 입력 작업이 번거로우셨나요?

이 도구는 선생님께서 미리 준비하신 교육 내용을 나이스(NEIS) 시스템 입력창에 편리하게 옮겨주는 간단한 보조 프로그램입니다.

---

## 🎯 이런 분들께 추천합니다

- 개별화교육계획 **월별계획** 작성 시 반복 입력이 부담스러운 선생님
- **월별평가** 내용을 빠짐없이 정확하게 기록하고 싶으신 선생님
- 엑셀이나 워드에 미리 작성한 내용을 나이스에 옮기는 작업이 번거로운 선생님
- **IEPON** 등 외부 도구로 작성한 계획을 나이스에 입력해야 하는 선생님

---

## ✨ 주요 기능

### 📋 월별계획 입력 지원
교육목표, 교육내용, 교육방법, 평가계획을 각 입력란에 정확하게 배치합니다.

### ✅ 월별평가 입력 지원
월별 평가 내용을 순번에 맞춰 정확하게 기록할 수 있도록 돕습니다.

### 🔍 자동 데이터 감지
붙여넣은 내용이 월별계획인지 월별평가인지 스스로 판단합니다.

### 📦 그룹 입력 지원
2~3개월 분량을 묶어서 한 번에 입력할 수 있습니다.

### 🔗 외부 도구 호환
IEPON 사이트나 다른 문서 작성 도구에서 준비한 내용을 바로 사용할 수 있습니다.

---

## 📖 사용 방법

1. **데이터 준비**: IEPON 사이트에서 작성한 내용을 복사하거나 JSON 파일을 준비합니다.
2. **나이스 이동**: 나이스(NEIS)의 개별화교육계획 메뉴에서 월별계획 또는 월별평가 화면으로 이동합니다.
3. **데이터 붙여넣기**: 확장 프로그램 아이콘을 클릭하고 데이터를 붙여넣습니다.
4. **입력 시작**: "입력 시작" 버튼을 누르면 각 항목이 해당 입력란에 채워집니다.

---

## 🔒 안전하게 사용하세요

이 프로그램은 선생님의 컴퓨터(브라우저) 안에서만 작동합니다.

- ✅ **모든 처리가 브라우저 내에서만 이루어집니다**
- ✅ **외부로 데이터를 전송하지 않습니다**
- ✅ **어떠한 정보도 저장하거나 수집하지 않습니다**
- ✅ **학생 정보나 교육 내용이 외부로 전송되지 않습니다**

## 🔐 권한 안내

이 확장프로그램이 요청하는 권한입니다:

| 권한 | 설명 |
|------|------|
| `activeTab` | 현재 보고 계신 탭에서 작동하기 위해 필요 |
| `storage` | 사용자 설정(입력 속도 등)을 기억하기 위해 필요 |
| `scripting` | 나이스 페이지의 입력란에 내용을 채우기 위해 필요 |
| `https://*.neis.go.kr/*` | 나이스 교육행정정보시스템에서만 작동 |

> 모든 권한은 프로그램 작동에 꼭 필요한 **최소한의 권한**만 요청합니다.
> 개인정보를 수집하거나 외부로 전송하지 않습니다.
