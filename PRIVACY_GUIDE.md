# 개인정보 보호 탭 작성 가이드 (수정됨)

나이스 시스템 관리 측의 오해를 피하기 위해, "자동화/매크로"보다는 **"입력 편의 도구"**라는 점을 강조하여 작성했습니다.

## 1. 전용 목적 설명
이 확장 프로그램은 교사가 '개별화교육계획(IEP)' 업무를 수행할 때 반복적인 텍스트 입력을 돕는 업무 편의 도구입니다. 사용자가 미리 준비한 교육 계획 및 평가 내용을 웹페이지의 입력 필드에 손쉽게 옮겨 적을 수 있도록 도와주며, 단순 반복 업무를 줄여 교사가 교육 활동에 더 집중할 수 있도록 지원합니다.

## 2. 권한 요청 이유

### activeTab 사용 근거
사용자가 확장 프로그램 아이콘을 클릭하여 기능을 실행할 때만, 현재 보고 있는 페이지의 입력 필드를 인식하고 사용자가 선택한 텍스트를 붙여넣기 위해 일시적으로 접근합니다.

### storage 사용 근거
사용자의 편의를 위해 자주 사용하는 설정값(예: 텍스트 입력 속도, 선호하는 입력 모드)이나 연동 사이트 주소를 사용자의 브라우저 내부에만 안전하게 저장하기 위해 사용합니다.

### scripting 사용 근거
사용자가 입력을 요청했을 때, 해당 페이지의 입력 필드 위치를 파악하고 텍스트를 올바르게 기입해 주기 위해 페이지 구조를 분석하는 역할을 합니다. (복사/붙여넣기 기능을 구현하기 위함)

### contextMenus 사용 근거
(사용하지 않는다면 '권한 없음'으로 처리 권장)
*작성 예시*: 사용자가 웹페이지의 텍스트를 드래그하여 IEP 입력 데이터로 바로 활용할 수 있도록 우클릭 메뉴 편의 기능을 제공합니다.

### notifications 사용 근거
입력 작업이 완료되었거나, 입력할 필드를 찾지 못했을 때 사용자에게 진행 상황을 알려주기 위해 사용합니다.

### alarms 사용 근거
(사용하지 않는다면 '권한 없음'으로 처리 권장)

### 호스트 권한 사용 근거
`https://dge.neis.go.kr/*`: 해당 교육 행정 시스템 페이지에서 입력 보조 기능이 올바르게 작동하도록 하기 위해 필요합니다.
`https://www.iepon.site/*`: 사용자가 자신의 데이터를 관리하는 외부 사이트에서 입력할 내용을 가져오기 위해 필요합니다.

## 3. 원격 코드 사용 여부
**아니요**, 원격 코드 권한을 사용하고 있지 않습니다.

## 4. 사용자 데이터 사용

*   **개인 식별 정보**: 체크 해제
*   **건강 정보**: 체크 해제
*   **금융 및 결제 정보**: 체크 해제
*   **인증 정보**: 체크 (로컬 저장)
    *   *설명*: 외부 데이터 연동을 위한 개인 식별 키를 사용자의 브라우저 로컬 스토리지에만 저장하며, 이는 로그인 편의를 위해서만 사용됩니다.
*   **개인적인 커뮤니케이션**: 체크 해제
*   **위치**: 체크 해제
*   **웹 기록**: 체크 해제
*   **사용자 활동**: 체크 (입력 보조)
    *   *설명*: 입력 보조 기능을 수행하기 위해 웹페이지 상호작용(클릭, 입력)을 수행합니다. 이 과정에서 수집되는 데이터는 없습니다.
*   **웹사이트 콘텐츠**: 체크 (텍스트 입력)
    *   *설명*: 입력 필드에 텍스트를 채워 넣기 위해 페이지 콘텐츠에 접근합니다.

**확인 사항 (모두 체크)**:
*   [v] 승인된 사용 사례를 제외하고 사용자 데이터를 제3자에 판매 또는 전송하지 않음
*   [v] 항목의 전용 목적과 관련 없는 목적으로 사용자 데이터를 사용하거나 전송하지 않습니다.
*   [v] 신용도 판단 또는 대출을 위해 사용자 데이터를 사용하거나 전송하지 않습니다.

## 5. 개인정보처리방침 URL
"본 확장 프로그램은 '입력 보조 도구'로서 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 모든 데이터 처리는 사용자의 컴퓨터 내에서만 이루어지며, 설정 정보는 브라우저에 안전하게 저장됩니다."
