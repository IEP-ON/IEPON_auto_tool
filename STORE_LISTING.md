# 크롬 웹 스토어 등록 정보 (수정됨)

## 1. 기본 정보
- **이름**: 나이스 월별계획 입력 도우미 (IEPON Helper)
- **요약**: 개별화교육계획(IEP) 월별 계획 및 평가 데이터 입력을 도와주는 업무 편의 도구입니다.

## 2. 상세 설명
**선생님의 업무 효율을 높여드립니다!**

나이스(NEIS) 시스템에서 개별화교육계획(IEP) 입력 업무가 번거로우셨나요?
'나이스 월별계획 입력 도우미'는 선생님이 준비한 내용을 손쉽게 입력할 수 있도록 도와주는 스마트한 보조 도구입니다.

### ✨ 주요 기능
*   **스마트 입력 보조**: 준비된 교육목표, 교육내용, 교육방법 텍스트를 클릭 한 번으로 입력 필드에 채워줍니다.
*   **평가 데이터 입력 지원**: 월별 평가 내용도 손쉽게 입력하고 저장할 수 있도록 도와줍니다.
*   **정확한 위치 인식**: 입력해야 할 행과 열을 정확하게 인식하여 오타나 입력 실수를 줄여줍니다.
*   **자연스러운 입력 방식**: 키보드로 직접 입력하는 방식을 시뮬레이션하여 시스템 호환성을 높였습니다.
*   **외부 데이터 연동**: JSON 파일이나 IEPON 사이트의 데이터를 간편하게 불러와서 활용할 수 있습니다.

### 🚀 사용 방법
1. IEPON 사이트에서 데이터를 가져오거나 JSON 파일을 준비합니다.
2. 나이스(NEIS)의 '개별화교육계획' 메뉴로 이동합니다.
3. 확장 프로그램을 열고 '입력 시작' 버튼을 누르세요.
4. 프로그램이 선생님을 대신해 텍스트를 입력해줍니다.

### 🔒 개인정보 보호
이 프로그램은 선생님의 컴퓨터(로컬)에서만 작동하는 입력 보조 도구입니다. 학생들의 개인정보를 외부 서버로 전송하거나 수집하지 않으므로 안심하고 사용하세요.
